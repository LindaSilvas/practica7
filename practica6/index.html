¿Qué es HTML?

HTML es un lenguaje de marcado que se utiliza para la creación de páginas web en la World Wide Web. Es el esqueleto de cualquier página web y estructura la información que se muestra.

¿Qué es CSS?

CSS es un lenguaje de diseño gráfico utilizado para definir y crear la presentación visual de documentos escritos en HTML o XML. Permite darle estilo y formato a las páginas web, controlando su apariencia.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web. Junto con HTML y CSS, es uno de los componentes clave para el desarrollo web moderno.
